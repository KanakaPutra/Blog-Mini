# Blog Laravel

Website blog ini dibangun menggunakan **Laravel** dan dirancang untuk mengelola artikel, kategori, komentar, dan pengguna dengan fitur otentikasi.

## Fitur
- CRUD artikel dan kategori
- Sistem komentar untuk artikel
- Registrasi dan login pengguna
- Manajemen role pengguna (admin & anggota)
- Tampilan responsif dan user-friendly
